# robots.txt - Block Google Crawling During Development
# Place this file in the root directory of your domain

# Block all crawlers from indexing the entire site
User-agent: *
Disallow: /

# ===== NOTES =====
# 
# This robots.txt file will:
# ✅ Prevent Google from indexing your website
# ✅ Keep your site out of Google Search results
# ✅ Allow you to work without worrying about SEO impact
#
# To DISABLE blocking (when your site is ready):
# Simply delete this file or modify Disallow to: Disallow: (empty)
#
# Location: Upload to root directory (same level as index.html)
# Example: chiangmai-home.com/robots.txt
#
# ===== TIMELINE =====
# 
# Phase 1 (Development - Now):
#   - Keep robots.txt with "Disallow: /"
#   - Google won't crawl
#   - Build your site freely
#
# Phase 2 (Ready to Launch):
#   - Delete robots.txt file
#   - OR change "Disallow: /" to "Disallow: " (empty)
#   - Google will start crawling 7-14 days later
#   - Your site appears in Google Search
#
# ===== THAI NOTES =====
#
# ไฟล์นี้ใช้เพื่อ:
# ✅ ป้องกัน Google indexing
# ✅ เว็บของคุณไม่แสดง Google Search
# ✅ สร้างเว็บได้ อย่างสบายใจ ไม่ต้องกังวล SEO
#
# ทำเสร็จแล้ว:
# ลบไฟล์นี้ หรือเปลี่ยน Disallow เป็นว่าง
# Google จะเริ่ม index 7-14 วันหลัง
#
